Install

インストール手順

MT4 へのセットアップは 3 ステップで完了します。

01
.ex4 をダウンロード

下部のダウンロードボタンから取得してください。

02
Experts フォルダに配置

MT4 → ファイル → データフォルダを開く → MQL4 / Experts へコピー。

03
チャートに適用

ナビゲーターから対象EAをチャートへドラッグ＆ドロップ。自動売買を許可してください。
