# インストール手順

MT4 へのインジケーター追加は 3 ステップで完了します。

1
.ex4 ファイルをダウンロード

上記ダウンロードボタンから .ex4 ファイルを保存してください。

2
MT4 の Indicators フォルダに配置

MT4 メニューの ファイル → データフォルダを開く をクリック → MQL4 / Indicators フォルダに .ex4 ファイルをコピーします。

3
インジケーターをチャートに適用

MT4 の「ナビゲーター」ウィンドウから対象インジケーターをチャートへドラッグ＆ドロップしてください。
