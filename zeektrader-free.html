# 概要

ZeekTraderFree は、裁量トレーダー向けの注文補助 EA です。

チャート上に表示されるパネルを操作するだけで、Buy / Sell の成行注文や指値注文の発注・決済がすぐに行えます。 ロットは手動で設定し、エントリー時のリスクリワードやドローダウン比率をリアルタイムに確認できます。 また、TP進行率や相場の勢い（ランウェイ）を自動判定して、ブレイクイーブン（SL を建値に移動）する機能も備えています。

🖱️
GUIパネルで即時注文 Buy / Sell ボタンで成行発注。指値・SL・TPもパネルから設定
⚖️
リスクリワード表示 SL・TP設定に連動してRR比をリアルタイム更新
📊
ドローダウン比率 累積損失額をもとに口座ドローダウン率を表示
🛡️

⏱️

📉
スプレッド計測 通貨ペアごとのスプレッドを継続計測・記録
📅

❌
指値TP到達時キャンセル 指値注文が TP 価格に到達した場合に自動でキャンセル
Free 版について： ロット管理は「手動入力」のみ対応です。資金比率・固定リスクによる自動ロット計算は製品版のみ対応しています。 バックテストには対応していません（リアルトレード専用）。

# パラメーター

EA のプロパティ画面から設定できる項目です。

表示設定
_THEME_COLOR パネルのテーマカラー（SOFT_BLUE など）
_ShowRemainingTime バー残り時間と変動pipsをチャート左下に表示するか（ON / OFF）
ロット管理
_LotManagerMethod Free 版は Manual（手動）のみ選択可能
リスク管理
_AutoBreakEvenByProgress TP進行率が 80% を超えた時点で自動的に SL を建値へ移動（ON / OFF）
_AutoBreakEvenByRunway H1 / H4 のランウェイが反転した場合に自動でブレイクイーブン（ON / OFF）
_CancelOnLimitReachTP 指値注文が TP 価格に到達した時点で自動キャンセル（ON / OFF）
_UseWeekEndRollover 週末前のポジション自動全決済を有効にするか（ON / OFF）
# インストール手順

MT4 への導入は 4 ステップで完了します。

1
ファイルをダウンロード

下記のダウンロードボタンから ZeekTraderFree.ex4 を保存してください。

2
MT4 の Experts フォルダに配置

MT4 メニューの ファイル → データフォルダを開く をクリック → MQL4 / Experts フォルダに ZeekTraderFree.ex4 をコピーします。

3
自動売買を許可する

MT4 の ツール → オプション → エキスパートアドバイザー で「自動売買を許可する」にチェックを入れてください。

4
チャートに適用

MT4 の「ナビゲーター」ウィンドウから ZeekTraderFree をチャートへドラッグ＆ドロップし、パラメーターを設定して OK をクリックします。
